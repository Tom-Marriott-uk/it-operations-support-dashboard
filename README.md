# IT Operations Support Dashboard

A Python desktop application that simulates an IT support operations workflow. The project includes support ticket logging, IT asset tracking, audit logging, dashboard metrics and CSV report export using SQLite.

## Project purpose

This project was built as a portfolio project to demonstrate practical skills relevant to IT support, technical support and digital operations roles.

Rather than creating a basic coding exercise, this project focuses on a realistic workplace process: recording support requests, tracking IT assets, keeping an audit trail and exporting operational reports.

## Features

- Create IT support tickets
- Record requester, location, category, priority and status
- Add issue descriptions and resolution notes
- Maintain an IT asset register
- Track asset condition and location
- View dashboard metrics
- Keep an audit log of key actions
- Export ticket, asset and audit records to CSV
- Uses a local SQLite database
- Runs as a desktop application

## Technologies used

- Python
- Tkinter
- SQLite
- CSV
- Standard Python libraries

## How to run

1. Download or clone this repository.
2. Open `it_operations_dashboard.py` in PyCharm.
3. Run the file.
4. Click **Load Sample Data** to populate the dashboard with example records.

No external packages are required.

This project demonstrates how Python can be used to support those kinds of operational tasks in a simple but practical way.

## Portfolio note

This is a portfolio project designed to simulate a small IT support operations system. It is not intended to be a full enterprise ITSM platform.
